Introduction

(WIP)

SymForce is an open-source library for writing algorithmic code like SLAM, MPC, calibration, bundle adjustment, and controls. Compared to alternatives it results in:

  1. Faster runtime performance, with a highly performant code generator
  1. Faster development cycles, using symbolic math with no handwritten derivatives
  1. Fewer lines of code, maintain one problem formulation and autogenerate the rest

The primary workflow of SymForce consists of three stages:

  1. Symbolic Math - Model your problem in Python with the SymPy API extended to include symbolic geometry types, camera models, and Lie group calculus
  1. Code Generation - Introspect symbolic functions to autogenerate highly performant runtime functions for any language
  1. Optimization - Use generated functions as factors in our tangent-space optimizer for real-time robotics applications, in either C++ or Python

This tutorial focuses on the use of symbolic computation, code generation, and nonlinear optimization to solve robotics problems using the recently released SymForce software library. During this interactive session, we will show users how they can use symbolic computation and code generation to reduce software development time without sacrificing computational performance on robotics problems such as SLAM, MPC, calibration, bundle adjustment, and controls.
