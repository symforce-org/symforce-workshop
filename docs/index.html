Symbolic Computation to Accelerate Robotics

A Hands-On Half-Day Tutorial at ICRA 2023

Introduction

This tutorial introduces the use of symbolic computation, code generation, and nonlinear optimization to solve robotics problems like SLAM, MPC, calibration, bundle adjustment, and controls with the recent SymForce software library.

This underused model of robotics development can result in faster runtime performance, faster development cycles, and fewer lines of code versus standard approaches, sometimes by an order of magnitude. During this interactive session, attendees will employ these techniques to solve a localization problem and an MPC problem, as well as deep dive into the underlying methods.

The SymForce library is used as a tool to demonstrate the following concepts:

  1. Symbolic Math: Problems are modeled symbolically in Python, making it easy to create differentiable symbolic functions using, e.g., 3D geometry and camera models. This allows introspecting and analyzing expressions separately from their runtime implementations.
  1. Code Generation: Symbolic functions are automatically converted into highly performant functions in other programming languages, such as C++. Function flattening and subexpression elimination reduce the number of unnecessary operations at runtime.
  1. Optimization — Generated functions are used in conjunction with different optimization algorithms to solve a variety of robotics problems. The primary focus will be on tangent-space nonlinear least squares using Levenberg-Marquardt.

Intended Audience

This tutorial is targeted at roboticists in both industry and academia interested in faster development of performant code for use in robotics applications. No prior knowledge of symbolic computation or the SymForce library is expected, though attendees should have some familiarity with programming in Python. Attendees are encouraged to bring laptops for interactive use. The learnings are broadly useful for both education and for production robotics use.
